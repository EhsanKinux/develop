این یک مخفف برای عبارت Immediately Invoked Function Expression هست. متغیرهایی که توی توابع تعریف میشن، دارای حوزه محلی (Local Scope) هستن. محصور کردن متغیرها توی یک تابع، ۲ خوبی داره:
1. متغیرها فقط مخصوص همون تابع یا اسکوپ هستن و از بیرون تابع قابل دسترسی نیستن. یعنی:

(function () {
const x = 10;
})()
alert(x); // ReferenceError: x is not defined;

2. متغیرهای گلوبال اسکوپ دستکاری نمی‌شن. یعنی:

const z = 20;
(function () {
const z = 10;
})()
alert(z); // 20

به روش زیر می‌تونیم به این توابع آرگومان پاس بدیم:

(function (x, y) {
alert(x + y)
})(10, 20)
